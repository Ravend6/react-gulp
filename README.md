# Gulp settings for React.js

### Gulp plugins
- reactify
- browserify
- stylus with autoprefixer
- imagemin
- server-livereload
- watch
- uglify
- minify-css
- minify-html

### Config project
Edit `gulpfile.js`
```js
    // Config
    var 
      isMinify = false,
      srcDir = './src/',
      distdDir = './dist/';
```

### Run project
###### Terminal:
-- `npm i` 
-- `gulp`
